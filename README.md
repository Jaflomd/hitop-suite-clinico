# Psiquiatria basada en medicion

Suite local-first de 40 instrumentos de cribado y severidad psicopatologica, pensada para uso clinico exploratorio en una sola shell web.

## Lo que hace

- Un instrumento por sesion
- Una pregunta por pantalla
- Reporte final con puntajes completos
- Traduccion dimensional / HiTOP-friendly
- Copia rapida de resumen y tablas
- Sin backend: todo corre localmente en el navegador

## Ejecutar local

```bash
python3 -m http.server 8010
```

Luego abre `http://127.0.0.1:8010`.

## Nota

La app es un apoyo de cribado y organizacion dimensional. No reemplaza entrevista clinica, juicio profesional ni evaluacion diagnostica formal.
