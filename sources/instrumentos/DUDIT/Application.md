# Test de Identificación de Trastornos por Consumo de Drogas (DUDIT) - Aplicación

**Instrucciones:** Por favor, responda a las siguientes preguntas sobre su consumo de drogas (distintas al alcohol). "Drogas" incluye sustancias ilegales, así como el uso de medicamentos de forma no prescrita o en dosis mayores a las indicadas.
