# Cuestionario de Ansiedad Generalizada (GAD-7) - Guía Técnica y Puntuación

## 🎯 Propósito
El GAD-7 es una herramienta breve de cribado diseñada para identificar casos probables de Trastorno de Ansiedad Generalizada (TAG) y medir la severidad de los síntomas de ansiedad.

---

## 📊 Interpretación y Puntuación
La puntuación total se obtiene sumando las respuestas a las 7 preguntas (de 0 a 3 cada una). El rango es de **0 a 21**.

### Niveles de Severidad:
| Puntuación | Severidad de la Ansiedad |
| :---: | :--- |
| **0 - 4** | Ansiedad mínima |
| **5 - 9** | Ansiedad leve |
| **10 - 14** | Ansiedad moderada |
| **15 - 21** | Ansiedad grave |

### Punto de Corte:
- Un puntaje de **≥ 10** es el punto de corte recomendado para identificar un posible Trastorno de Ansiedad Generalizada.
- También es útil para detectar trastorno de pánico, trastorno de ansiedad social y trastorno por estrés postraumático (aunque con menor especificidad).

---

## ⚖️ Limitaciones Importantes
- **Cribado, no Diagnóstico:** Al igual que el PHQ-9, es una medida de severidad y cribado. Un puntaje alto debe ser seguido por una evaluación clínica.
- **Transdiagnóstico:** Los síntomas de ansiedad suelen coexistir con la depresión; se recomienda aplicar junto con el PHQ-9.
